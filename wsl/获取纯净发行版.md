[Index of /wsl/releases](https://cloud-images.ubuntu.com/wsl/releases/)
从以上连接中, 选择下载amd64的`.rootfs.tar.gz`文件
然后执行命令导入, 注意新建文件夹
```powershell
wsl --import ubuntu-cuda E:\linux\ubuntu-cuda E:\linux\ubuntu-noble-wsl-amd64-24.04lts.rootfs.tar.gz
```
由于开启了默认稀疏, 可能会提示
```powershell
wsl: 由于潜在的数据损坏，目前已禁用稀疏 VHD 支持。=========]
若要强制分发使用稀疏 vhd，请运行：
wsl.exe --manage <DistributionName> --set-sparse --allow-unsafe
```
此时需要
```powershell
wsl --shutdown
wsl.exe --manage ubuntu-cuda --set-sparse true --allow-unsafe
```
由于是纯净版, 所以需要创建用户, 并设置默认用户
```bash
useradd -m -G sudo -s /bin/bash wsl
passwd wsl
# 设置为 789

echo -e "[user]\ndefault=wsl" > /etc/wsl.conf
exit
```
然后执行
```powershell
wsl --terminate ubuntu-cuda
```
然后重新进入就可以了
设置镜像

```bash
sudo sed -i 's@//.*archive.ubuntu.com@//mirrors.ustc.edu.cn@g' /etc/apt/sources.list.d/ubuntu.sources
```
